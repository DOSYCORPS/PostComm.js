PostComm.js Unit Tester

PostComm.js is a javascript microlibrary designed to handle the security and routing of postMessage events for mass numbers of iframes and windows.

Find out more at the repository: https://github.com/dwighthouse/PostComm.js

This page was created to help run the unit tests online, without requiring you to download the source or set up your own servers.

Popup blockers will prevent window-based tests from functioning. Please disable these before running the tests.
